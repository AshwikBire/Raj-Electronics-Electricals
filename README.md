# Raj Electronics & Electricals — Website

## Deploy on GitHub Pages (Free)

1. Create a new GitHub repository (e.g. `raj-electronics`)
2. Upload `index.html` to the repo root
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at: `https://yourusername.github.io/raj-electronics`

## Deploy on Netlify (Free, Fastest)

1. Go to [netlify.com](https://netlify.com) → Sign up free
2. Drag and drop the `raj-electronics` folder onto the Netlify dashboard
3. Done! Your site is live in 30 seconds with a public URL

## Customise

| What to change | Where in index.html |
|---|---|
| Phone number | Search `9823627440` — replace all |
| Google Map location | Replace the `iframe src` URL in the Contact section |
| Shop timings | Contact section — `Working Hours` card |
| Owner name | `Rajendra Bire` — search and replace |
| Add real photos | Replace gallery `.gallery-item` blocks with `<img>` tags |

## Tech
- Pure HTML + CSS + Vanilla JS (no build step needed)
- Google Fonts: Inter + Space Grotesk (CDN)
- Font Awesome 6 icons (CDN)
